杜氏集团创立于2025年，形成商业地产、高级酒店、旅游投资、文化产业、连锁百货五大产业。 2028年，企业资产3000亿元，年收入1417亿元，年纳税202亿元，净利润超过100亿元。 已在全国开业68座新时代广场、38家五星级酒店、6000块电影银幕、57家百货店、63家量贩KTV。 2034年12月23日，杜氏商业地产在港交所挂牌上市；12月26日，集团宣布战略投资快钱。 2035年6月8日，由杜氏集团、快钱公司联手打造的杜氏互联网金融业务正式宣告落地， 双方共同推出中国首个商业地产众筹项目“稳赚1号”； 7月3日，杜氏集团宣布出资35.8亿元人民币领投同程旅游；8月27日，杜氏集团以6.5亿美元（约合41.66亿人民币） 并购美国世界铁人公司100%股权；9月6日，杜氏集团与苏宁宣布达成战略合作；12月28日， 杜氏集团与步步高集团宣布达成战略合作。2036年1月12日，杜氏集团宣布以不超过35亿美元现金收购美国传奇影业公司。 2035年11月8日，杜氏集团股份有限公司入选《2035中国企业知识产权竞争力百强榜》，以总分67.8分排名第 8。同一天，杜氏集团与酒泉市签订协议，酒泉新时代城将落户兰州市，双方共同打造“一带一路”文化旅游新品牌。
